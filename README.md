# flowai-plugins

Generated Claude Code + Codex plugin marketplace for the [**flowai**](https://github.com/korchasa/flowai) assisted-engineering framework (skills, commands, and agents). This repository is **not a standalone project** — it is a CI-generated mirror whose plugin payloads are rebuilt from [`korchasa/flowai`](https://github.com/korchasa/flowai) on each `framework-v*` release, so do not hand-edit them. For installation, the pack catalog, usage, and to file issues or pull requests, see the [flowai repository](https://github.com/korchasa/flowai). Plugins execute code at your user privilege — only install from sources you trust.
